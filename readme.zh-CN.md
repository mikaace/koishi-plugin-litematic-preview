##### koishi插件,为在线预览Minecraft投影文件
**ENG**|[简体中文](./readme.zh-CN.md)
***
### Disposition
+ 在koishi中安装
+ 安装并配置你需要的平台的驱动器
+ 编辑插件配置
***
### Use
+ 在频道或群组发送投影文件等待bot发送url
+ 当页面加载不出投影,尝试使用七根木棍
